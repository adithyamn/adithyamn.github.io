3D Cartesian Manipulator Modeling and Control

❮ ❯

Project Overview

This project explores and compares two advanced control strategies for a 3-DOF Cartesian manipulator: a Force-Based Proportional-Derivative (PD) controller and Nonlinear Model Predictive Control (NMPC) using Acados. The aim is to achieve precise and efficient trajectory tracking through complex multi-point paths in 3D space.

It features a detailed dynamic robot model, quintic polynomial trajectory generation for smooth motion, and the implementation of both PD and NMPC for commanding forces. Comprehensive analysis includes 3D visualizations of actual and desired trajectories, alongside 2D plots detailing position/velocity tracking errors, commanded forces, power consumption, and cumulative energy.

This work highlights the strengths of reactive (PD) and predictive (NMPC) control in complex robotic motion scenarios.
